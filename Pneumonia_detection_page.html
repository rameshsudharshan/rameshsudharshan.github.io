Ensemble Deep Learning for Pneumonia Detection

Project Overview

Developed an ensemble deep learning model for pneumonia detection, combining ResNet-18, DenseNet-121, and GoogLeNet with a hyperbolic tangent-based weighting mechanism to enhance classification accuracy.

This model leverages the RSNA Pneumonia Detection Challenge dataset, applying preprocessing steps like image normalization and augmentation to improve robustness against variations in X-ray scans.

Model Architecture & Results

Each CNN model in the ensemble independently processes input X-ray images, producing probability scores for pneumonia detection. These scores are then weighted using a hyperbolic tangent function based on key performance metrics (precision, recall, AUC-ROC).

The final classification decision is derived from the weighted outputs, achieving a 6.3% increase in F1-score and a 9.1% improvement in sensitivity compared to the best-performing individual CNN model. The ensemble approach enhanced overall classification accuracy by 5.7%, particularly improving detection performance on hard-to-classify cases in the dataset.
