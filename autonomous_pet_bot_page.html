Autonomous Pet Bot

Project Overview

Engineered a robotic pet with sensor-driven navigation, multi-actuator mobility, and ROS2-based integration. Utilizing LiDAR and Intel RealSense camera, the pet achieves enhanced SLAM-based localization, gesture recognition, and autonomous mapping.

Design Features

  • Wagging Tail: 4-bar linkage mechanism
  • Legs with Wheels: Hybrid movement strategy
  • Microphone & Speaker: Speech recognition & response
  • Camera Vision: Real-time object tracking & recognition using YOLOv7
  • IMU & Lidar Fusion: High-frequency Kalman filtering for noise reduction

ROS2 Integration & Sensor Fusion

The robotic pet uses a ROS2-based software stack for real-time control. A high-frequency Kalman Filter fuses data from:

  • LiDAR - Distance tracking
  • RealSense Camera - Object & face detection
  • IMU - Motion & balance correction

ROS2 Nav2 package is implemented for path planning and localization, allowing the pet to navigate autonomously in dynamic environments.
