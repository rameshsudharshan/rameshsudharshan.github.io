IoT-Based Constant Speed Propeller System

Project Overview

This IoT-based constant speed propeller system leverages real-time RPM monitoring and motor control. The system ensures stable propeller speed despite varying load conditions by implementing closed-loop control.

The system integrates:

- Arduino UNO WiFi REV2 microcontroller for wireless connectivity.

- RPM sensor for real-time speed feedback.

- PID control loop for maintaining desired speed.

- MQTT communication for remote control and monitoring.

Performance Results

The system effectively maintains the set RPM, adapting to external disturbances.

- Fast response time for speed adjustments.

- Minimal overshoot during transient changes.

- Stable steady-state RPM within a 2% error margin.
