Automated Testing with UR5 Robot

Project Overview

Developed an automated testing system for evaluating pneumatic actuator performance using a 6-axis UR5 robotic arm. The robotic system utilizes inverse kinematics to position the load cell at precise locations to measure force at different actuator positions.

The testing process is fully automated, eliminating manual errors and ensuring consistent force measurements across multiple tests.

Inverse Kinematics Implementation

The UR5 arm's motion is controlled through inverse kinematics algorithms, ensuring precise end-effector positioning. The trajectory is optimized to maximize the accuracy of force measurements while minimizing unnecessary movement.
