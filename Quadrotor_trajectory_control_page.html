Quadrotor Trajectory Control System

Project Overview

Designed a PID and Model Predictive Control (MPC) based trajectory control system for a quadrotor drone, enhancing trajectory tracking under wind disturbances. Using MATLAB Simulink, this approach achieved a 40% improvement in stability metrics.

The quadrotor simulation model incorporates realistic wind gust effects, sensor noise, and system dynamics. Our control strategy utilizes adaptive gains to dynamically adjust response time based on flight conditions.

Control System & Results

Our control system consists of a dual-loop PID architecture, where an outer loop handles position control and an inner loop manages attitude stabilization. Model Predictive Control (MPC) further enhances adaptability by predicting future trajectory deviations and adjusting control inputs accordingly.

Simulated results demonstrate effective stabilization under wind disturbances, with significant improvements in tracking accuracy. The controller was tested across multiple trajectory profiles, showcasing real-time corrections and optimized thrust allocation.
