Autonomous Battle Bot

Project Overview

The Autonomous Ball Shooter Robot was developed to demonstrate real-time motion control and robotic actuation. It integrates computer vision and mechanical design to achieve a fully automated shooting mechanism.

Project Design

The system was designed using modular engineering principles, allowing for easy assembly, disassembly, and future upgrades. The structure consists of a motorized base equipped with Mecanum wheels, enabling omnidirectional movement for precise positioning.

A robust sensor mounting system ensures reliable integration of vision sensors and IR sensors for real-time environmental awareness. The shooting mechanism is driven by high-speed motors and a precision indexing system, which allows for consistent and accurate ball firing.

The automated reloading mechanism features a vertical ball silo that feeds the shooter via a controlled release system, ensuring a seamless operation without manual intervention. Designed with 3D-printed parts, the robot is both lightweight and structurally rigid, providing a balance between speed and durability.

The electronics and control system are housed in a dedicated compartment to protect against external interference, ensuring smooth data processing and power distribution. By adopting a modular framework, this robot allows for easy troubleshooting, maintenance, and potential expansions in functionality.

Electronics and Circuit Design

The electronic system is powered by an Arduino Mega, chosen for its high number of I/O pins and efficient processing capability. It serves as the central controller, managing motor actuation, sensor data acquisition, and communication between various subsystems.

To ensure precise motion control, the Arduino interfaces with motor drivers that regulate the power delivered to the Mecanum wheels. These drivers optimize speed and torque while preventing current spikes that could damage the circuit.

Ball Reloading Mechanism

The reloading system features a vertical silo that holds multiple balls, ensuring a continuous supply to the shooter. A sliding release mechanism at the base allows controlled ball delivery.

When activated, the sliding gate moves, briefly opening a passage for a single ball to roll into the shooter before closing again. This mechanism prevents multiple balls from entering at once and ensures smooth operation.

The silo is designed with smooth internal surfaces to prevent jams and ensure balls roll down freely. By automating this process, the system enables **continuous shooting** without manual intervention.
